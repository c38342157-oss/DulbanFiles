03 · 3D-прогиб при наведении

Води мышкой: кнопка «прогибается» в 3D — угол под курсором уходит вглубь экрана (press-режим), а карточка справа — классический tilt с параллакс-слоями и бликом.

press: true — прогибается под курсором
Параллакс TILT CARD

Слои с data-tilt-depth всплывают над поверхностью.

NEON TILT

На тёмном блик становится неоновым (mix-blend-mode: screen).

Код 
// кнопка прогибается под мышкой (угол уходит ВГЛУБЬ)
DulbanTilt.attach(btn,  { max: 12, press: true,  glare: true, sink: 12 });

// классический tilt: угол тянется К курсору, дети с data-tilt-depth парят
DulbanTilt.attach(card, { max: 10, press: false, glare: true });

// или автоматом для всех <div data-db-tilt>
DulbanTilt.init();

04 · Кривые плавности

График каждой кривой построен из реальной функции DulbanEase. Кликни по карточке — точка пробежит по графику, а шарик внизу проедет с этой кривой. В CSS те же кривые доступны как var(--ease-out-cubic) и т.д.

Код 
/* CSS */
.menu { transition: transform .45s var(--ease-out-back); }

/* JS */
const e = DulbanEase.get('inOutCubic');   // t -> прогресс
DulbanTween.to(el, { duration: 600, ease: 'outExpo', x: 240 });
DulbanEase.css.outExpo;                   // 'cubic-bezier(.16,1,.3,1)'

05 · Tweening как в After Effects

DulbanTween — мини-движок на requestAnimationFrame: ключевые кадры со своими кривыми на каждый сегмент (как ключи в AE), таймлайн с оффсетами '-=200', stagger, yoyo, repeat.

Код-шаблон 
// простой твин (аналог двух ключей в AE)
DulbanTween.to(box, { duration: 900, ease: 'inOutCubic', x: 420 });

// yoyo — туда и обратно
DulbanTween.to(box, { duration: 350, ease: 'outQuad', scale: [1, 1.35], yoyo: true, repeat: 1 });

// ключевые кадры: t — доля времени, v — значение, ease — кривая подхода к ключу
DulbanTween.keyframes(box, {
  prop: 'x', duration: 1600,
  frames: [
    { t: 0,    v: 0 },
    { t: 0.35, v: 420, ease: 'outExpo'   },  // резкий вылет
    { t: 0.65, v: 340, ease: 'inOutQuad' },  // откат
    { t: 1,    v: 420, ease: 'outBack'   },  // дожать с перелётом
  ],
});

// таймлайн — как слои в AE, '-=200' = начать на 200мс раньше конца предыдущего
DulbanTween.timeline()
  .add(box, { duration: 500, ease: 'outExpo',  y: [0, -20] })
  .add(box, { duration: 500, ease: 'outBounce', y: 0 }, '-=120')
  .add(box, { duration: 700, ease: 'inOutBack', rotate: 360 })
  .play();
