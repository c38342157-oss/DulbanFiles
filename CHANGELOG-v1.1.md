# Dulban Menu v1.1.0

## Изменения

- Исправлена индикация активных переключателей:
  - активная дорожка становится чёрной только внутри тумблера;
  - бегунок остаётся белым и перемещается вправо;
  - состояние дублируется через `aria-pressed` и `.is-on`.
- GitHub-лоадер теперь подключает весь Dulban Lib из корня репозитория.
- Добавлена строгая последовательность JS-зависимостей: easings → tween → tilt → SVG → menu.
- Автоматически внедряются `dulban-shapes.svg` и `dulban-icons.svg`.
- `DulbanTween` используется для появления панели и вкладок.
- `DulbanTilt` используется для FAB-кнопки и карточек статистики.
- `DulbanSVG` используется для молнии в кнопке запуска.
- Добавлен безопасный фолбэк: основное меню работает даже при недоступности декоративной библиотеки.
- README исправлен под реальную плоскую структуру репозитория.
