## Найденные и исправленные дефекты

1. Захват редкостей ожидал `common` первым элементом, но FrostDrop начинает массив с `the_shit`. Исправлено на поиск `common` по всему массиву.
2. CSS `display:flex` перекрывал HTML-атрибут `hidden`, и панель могла быть видна после запуска. Добавлено `.dm-panel[hidden]{display:none!important}`.
3. Синий акцент не соответствовал строгому W&B. Заменён на чёрный; зелёный/красный оставлены только как функциональные состояния success/danger.
4. GitHub-лоадер настроен на `c38342157-oss/DulbanFiles`.
5. Добавлены PNG-иконки 48/96 для Firefox manifest.

## Ожидаемые сообщения тестовой среды

Headless Chromium без сети сообщает `ERR_NAME_NOT_RESOLVED` для Google Fonts и `ERR_FILE_NOT_FOUND` для локальных ресурсов FrostDrop. Это не ошибки кода меню; при наличии сети/исходных ассетов браузер загружает их штатно.
